{
https://dev.mysql.com/doc/mysql-apt-repo-quick-guide/en/
Generated by IA:
1. Atualizar os Repositórios
Antes de qualquer instalação, é fundamental garantir que a lista de pacotes do seu sistema esteja atualizada.

Bash
sudo apt update
2. Instalar o Servidor MySQL
O comando abaixo instala a versão estável mais recente disponível nos repositórios oficiais da sua distribuição.

Bash
sudo apt install mysql-server
Durante a instalação, o sistema baixará e configurará os arquivos necessários automaticamente.

3. Verificar o Status do Serviço
Após a instalação, o MySQL deve iniciar automaticamente. Você pode conferir se ele está rodando com o comando:

Bash
sudo systemctl status mysql
Dica: Se o serviço não estiver ativo, use sudo systemctl start mysql.

4. Configuração de Segurança (Essencial)
O MySQL vem com uma instalação padrão que não é segura. Existe um script interativo que remove usuários anônimos e restringe o acesso remoto ao root.

Execute:

Bash
sudo mysql_secure_installation
O que o script vai perguntar:

VALIDATE PASSWORD COMPONENT? (Deseja validar a força das senhas?) – Digite y se quiser senhas complexas ou n para senhas simples.

Nova senha para o root: Defina uma senha mestre segura.

Remove anonymous users? Digite y.

Disallow root login remotely? Digite y.

Remove test database? Digite y.

Reload privilege tables now? Digite y.

5. Acessando o MySQL
Para entrar no console do banco de dados como administrador:

Bash
sudo mysql -u root -p
//
show databases;

}
